## Rename
For the indecisive developer

[![Build Status](https://travis-ci.org/MainShayne233/rename.svg?branch=master)](https://travis-ci.org/MainShayne233/rename)

## Usage
Add to your mix dependencies in `mix.exs`
```elixir
# mix.exs
defp deps do
  [
    {:rename, "~> 0.0.1"}
  ]
end
```
and install
```bash
mix deps.get
```
